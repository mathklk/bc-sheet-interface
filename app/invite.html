Google Sheets Integration - Einladungslink Generator

Anleitung:
  1. Spreadsheet ID: Die ID aus der Google Sheets URL (der Teil nach /d/ und vor /edit)
  2. Sheet Name: Der Name des Tabs/Blatts in der Tabelle
  3. API Key: Dein Google Sheets API Key

Die Tabelle sollte in Zeile 5 beginnen mit den Spalten: SuggestedBy, Funfacts, Buchtitel, Autor, Jahr, Seitenzahl, Genres, Kurzbeschreibung

Beispiel URL: https://docs.google.com/spreadsheets/d/1BxiMVs0XRA5nFMdKvBdBZjgmUUqptlbs74OgvE2upms/edit
Kopiere nur den fett markierten Teil (die ID)
